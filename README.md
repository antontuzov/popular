# Popular
:rocket: A Sample application written in [Kotlin][kotlin] and using [Anko][anko].


## Description
- Use popular data
    - https://api.instagram.com/v1/media/popular?client_id=`<CLIENT-ID>`


## Pre-conditions
- Install Kotlin IDE Plugins
    - Kotlin
    - Anko DSL Preview (optional)


## Features
- [Kotlin][kotlin]
- [Rxjava][rxjava]
- [Retrofit][retrofit]
- [anko][anko]
